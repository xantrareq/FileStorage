# CloudStorage
# Система удаленного хранения файлов.

### Это веб-приложение, которое позволяет вам хранить свои файлы на удаленном сервере, а также делиться ими с другими пользователями

## Возможности

- **Регистрация и аутентификация**: У вас есть свой личный профиль)
- **Загрузка файлов**: Загружайте свои файлы.
- **Создание папок**: Создавайте свои директории для более удобной работы.
- **Обмен файлами**: Вы можете создавать ссылку на файл, которая будет действовать ограниченное количество времени.
